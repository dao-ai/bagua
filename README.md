# ☰ 八卦入门

每天15分钟，搞懂八卦。

一个交互式的八卦学习网站，纯前端，浏览器直接打开就能用。

## 功能

- **八卦学院** — 八个卦的卡片展示 + 详情弹窗
- **二进制对照** — 十进制 7→0，爻象 + 二进制码
- **六十四卦** — 全部64卦浏览，点击查看卦辞·大象·简释
- **起卦工具** — 数字法起卦，显示本卦→变卦 + 卦辞解读
- **变爻演示** — 动爻闪烁翻转动画
- **双主题** — 暗色/古风切换，自动保存偏好

## 使用

浏览器直接打开 `index.html`，无需任何安装或依赖。

## 技术栈

纯 HTML + CSS + JavaScript，零依赖。
